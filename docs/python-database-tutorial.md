## Python MongoDB

MongoDB 是最流行的 NoSQL 数据库之一。它是一个跨平台、面向对象的数据库。基本上，NoSQL 意味着 MongoDB 不以表或关系格式存储数据，而是提供不同的数据存储和检索机制。这叫做 BSON，类似于 JSON。这就是为什么 MongoDB 提供高速、高可用性和高可扩展性。
