## Python 和 Bash 的区别

*   **定义:**
    Python 是一种高级编程语言，旨在易于阅读和实现。而 Bash 是一个 sh 兼容的命令语言解释器，它执行从标准输入或文件中读取的命令。
*   **简约:**
    蟒蛇更容易维护。而 bash 不需要，它不需要维护。
*   **性能:**
    Bash 是你所知道的每一个 Linux 发行版以及 macOS 上的默认用户外壳，这使得它在性能上相对比 Python 更快
